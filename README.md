IIoT Utility Monitoring Dashboard

Overview

A virtual Industrial Utility Monitoring System developed in a home-lab environment to simulate real-world facility monitoring and control architectures used in manufacturing plants and mission-critical infrastructure.

The project demonstrates practical implementation of industrial telemetry, time-series data processing, and operational visualization using modern IIoT technologies.

⸻

Technology Stack





Node-RED — Data acquisition and workflow automation



MQTT (Mosquitto) — Industrial messaging protocol



InfluxDB — Time-series database



Grafana — Real-time monitoring and visualization



Docker — Containerized deployment



Ubuntu Linux — Edge computing platform

⸻

Key Features

PUMP-01 Utility Monitoring





Real-time motor current monitoring



Power consumption trend analysis



Equipment run/fault status visualization



Alarm indication and severity management



Total operating hours calculation



Start count tracking for maintenance planning

AHU-01 HVAC Monitoring





OA / RA / SA / Room temperature monitoring



Supply fan frequency monitoring



Return fan frequency monitoring



Power consumption tracking



Filter alarm monitoring



HVAC abnormal condition visualization
